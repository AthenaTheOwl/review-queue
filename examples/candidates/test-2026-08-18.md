# Test case: provenance filter rejects bare scheme

A regression test for the scoring loader: a promotion-record whose
`provenance_ref` is a bare scheme (`trace:` without `//`) must be
dropped from the calibration set, not scored. Without this the loader
silently counts malformed rows.

```python
def test_loader_drops_bare_scheme_provenance():
    rec = make_record(provenance_ref="trace:run/1")  # missing //
    assert not score._row_is_eligible(rec)
```
